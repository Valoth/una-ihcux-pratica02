# Projeto Olá Mundo em .NET

Este projeto tem como objetivo realizar o primeiro contato prático com a plataforma .NET, utilizando a criação de uma aplicação simples de console.

A atividade consiste em criar, executar e modificar um programa básico que exibe uma mensagem na tela.

## Objetivo

- Criar um projeto utilizando a CLI do .NET  
- Executar um programa de console  
- Compreender a estrutura inicial de um projeto  

## Criação do Projeto

O projeto foi criado utilizando o comando:

`dotnet new console -n OlaMundo`

Após a criação, foi necessário acessar a pasta:

`cd OlaMundo`

## Execução

Para executar o projeto:

`dotnet run`

Saída inicial padrão:

Hello, World!
